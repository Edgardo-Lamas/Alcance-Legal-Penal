1. Qué es MEV Navigator

MEV Navigator es una extensión de Chrome desarrollada por Studio Lamas como parte del sistema Alcance Legal Penal. Su función es asistir a abogados defensores en el análisis de expedientes judiciales disponibles en la Mesa de Entradas Virtual de la Suprema Corte de Buenos Aires (mev.scba.gov.ar).

La extensión opera exclusivamente en el dominio mev.scba.gov.ar. No tiene acceso a ningún otro sitio web ni a datos de otras pestañas del navegador.

2. Qué datos recopila la extensión

✅ Resumen ejecutivo: MEV Navigator no recopila, no almacena ni envía datos a servidores propios de Alcance Legal Penal. Todos los datos permanecen en tu dispositivo o se envían directamente desde tu navegador a los servicios que vos mismo configuraste (Anthropic API).
Tipo de dato ¿Se recopila? Dónde queda
Contenido del expediente judicial (carátula, actuaciones) Solo en memoria RAM del navegador durante la sesión. No se persiste.
API Key de Anthropic Local solamente chrome.storage.local — en tu navegador. Nunca sale del dispositivo.
Historial de análisis (últimas 20 causas) Local solamente chrome.storage.local — en tu navegador. Podés borrarlo en cualquier momento.
Preferencias de configuración (modelo IA, auto-análisis) Local solamente chrome.storage.local — en tu navegador.
Credenciales del MEV (usuario/contraseña SCBA) No se recopila La extensión NUNCA accede ni almacena tus credenciales del MEV.
Datos de navegación, historial de Chrome No se recopila La extensión solo opera en mev.scba.gov.ar. No tiene acceso a otras pestañas.
Datos de telemetría o analítica No se recopila No hay tracking, analytics ni telemetría de ningún tipo.

3. Datos del MEV: cómo se procesan

Cuando abrís una causa en el MEV, la extensión lee el contenido HTML de esa página para extraer la carátula (número de expediente, partes, organismo) y la lista de actuaciones. Este proceso ocurre localmente en tu navegador.

Flujo de datos cuando hacés "Analizar expediente con IA":
  1. Los datos del expediente se formatean como texto en tu navegador.
  2. Tu navegador envía ese texto directamente a los servidores de Anthropic usando tu propia API Key.
  3. Anthropic devuelve el análisis a tu navegador.
  4. El análisis se muestra en el panel lateral y, opcionalmente, se guarda en chrome.storage.local de tu navegador.

En ningún paso de este flujo los datos pasan por servidores de Alcance Legal Penal.

El tratamiento de datos enviados a Anthropic está regido por la Política de Privacidad de Anthropic. Te recomendamos revisarla. Por defecto, Anthropic no usa los datos de la API para entrenar modelos, salvo consentimiento explícito.

4. Uso de chrome.storage

La extensión utiliza chrome.storage.local y chrome.storage.session exclusivamente para:

  • Tu API Key de Anthropic — guardada de forma cifrada por Chrome en tu perfil local.
  • Preferencias (modelo seleccionado, opción de auto-análisis).
  • Historial local de los últimos 20 análisis realizados (carátula + texto del análisis). Podés borrarlo en cualquier momento desde la pestaña "Historial".
  • Datos de sesión temporales (session) del expediente activo — se eliminan automáticamente al cerrar Chrome.

Estos datos nunca salen de tu dispositivo hacia servidores de Alcance Legal Penal.

5. Llamadas a la API de Anthropic

Al usar la función de análisis, tu navegador realiza una llamada HTTP directa a api.anthropic.com usando la API Key que vos configuraste. Esta comunicación:

  • Usa HTTPS (cifrado en tránsito).
  • Incluye el texto del expediente como parte del prompt.
  • Está sujeta a los Términos de Uso de Anthropic.
  • Es responsabilidad del usuario no enviar información clasificada o sujeta a secreto sumarial.
Nota beta: En esta versión beta, las llamadas se realizan directamente desde el navegador. En versiones futuras, las llamadas pasarán por el backend autenticado de Alcance Legal Penal, eliminando la necesidad de que el usuario gestione su propia API Key.

6. Lo que NO hacemos

  • ❌ No enviamos datos de expedientes a servidores de Alcance Legal Penal.
  • ❌ No vendemos, alquilamos ni compartimos tus datos con terceros.
  • ❌ No accedemos a tus credenciales del MEV (usuario/contraseña SCBA).
  • ❌ No tenemos acceso a los datos guardados en chrome.storage (solo el navegador y la extensión pueden leerlos).
  • ❌ No rastreamos tu actividad de navegación fuera de mev.scba.gov.ar.
  • ❌ No usamos tus datos para entrenar modelos de IA.
  • ❌ No mostramos publicidad de ningún tipo.

7. Seguridad

Tomamos las siguientes medidas para proteger tus datos:

  • La API Key se almacena en chrome.storage.local, que Chrome cifra usando las credenciales de tu perfil del sistema operativo.
  • La extensión usa Manifest V3, que limita los permisos y el acceso al sistema respecto a versiones anteriores.
  • Los permisos de host están restringidos exclusivamente a mev.scba.gov.ar.
  • No se inyecta código externo ni se cargan scripts desde CDNs externos.

Si encontrás una vulnerabilidad de seguridad, por favor reportala a seguridad@alcancelegal.com.ar antes de divulgarla públicamente.

8. Menores de edad

Esta extensión es una herramienta profesional destinada exclusivamente a abogados y profesionales del derecho. No está dirigida a menores de 18 años. No recopilamos intencionalmente información de menores.

9. Cambios a esta política

Podemos actualizar esta política de privacidad cuando sea necesario. Los cambios significativos serán notificados a través de la descripción de la extensión en Chrome Web Store o mediante las notas de la versión. La fecha de "Última actualización" al inicio de este documento siempre indicará cuándo fue la revisión más reciente.
