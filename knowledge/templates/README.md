# Templates de Escritos

Esta carpeta contiene modelos de escritos judiciales para la capacidad de redacción.

## Estructura

- `demandas/` - Modelos de demandas por tipo
- `contestaciones/` - Modelos de contestaciones
- `recursos/` - Modelos de recursos (apelación, queja, etc.)

## Uso en el Sistema

Los templates sirven como base estructural. El sistema:
1. Selecciona el template apropiado
2. Adapta la estructura al caso
3. Genera contenido específico
4. Señala secciones pendientes

**IMPORTANTE**: Los templates son puntos de partida, no documentos finales.
