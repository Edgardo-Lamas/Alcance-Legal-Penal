# Jurisprudencia Curada

Esta carpeta contiene extractos de jurisprudencia curada para referencias del sistema.

## Criterios de Selección

- Jurisprudencia de Cámaras Nacionales (CNCiv, CNCom)
- Fallos de CSJN relevantes
- Decisiones de últimos 5 años (preferentemente)
- Casos emblemáticos históricos

## Formato

Cada documento debe incluir:
- Carátula completa
- Fecha del fallo
- Tribunal
- Sumario/extracto relevante
- Doctrina aplicable
