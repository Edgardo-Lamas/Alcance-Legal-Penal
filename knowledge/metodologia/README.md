# Metodología de Análisis

Esta carpeta contiene los marcos metodológicos utilizados por el sistema para analizar casos.

## Estructura

- `analisis/` - Esquemas de análisis jurídico
- `auditoria/` - Frameworks de auditoría estratégica  
- `redaccion/` - Pautas de redacción de escritos

## Principios

1. **Objetividad**: Análisis basado en elementos declarados
2. **Prudencia**: Siempre señalar incertidumbres
3. **Claridad**: Explicaciones entendibles
4. **Fundamento**: Citas verificables
